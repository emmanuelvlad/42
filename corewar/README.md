## Requirements

**ncurses** (only if using -nc option)  
Arch based distro:  
`sudo pacman -S ncurses`

## Usage

Compile `make`  
Assemble champions with `./asm champions/name.s`  
Start the war `./corewar -n 1 champions/name.cor -n 2 champions/name.cor -nc`
